# dotfiles

Modified from [mathiasbyens/dotfiles](https://github.com/mathiasbynens/dotfiles)

1. Run `./setup.sh`
2. Run `./brew.sh`
3. Run `./bootstrap.sh`

## Preference Files

Preferences files are located in `.init`.

### Divvy

1. Shut down Divvy
2. Copy preferences file to `~/Library/Preferences`
3. Reboot
